# Docker-Microservice-CRUD

## Description
A project to comprehend about basic microservice automation and deployment. Learn about create, read. edit, and delete files from a websites using API. You can use this repo to learn more and having fun :D

## Features
- CRUD API
- PostgreSQL
- Docker
- MinIO file storage

## Installation
```bash
git clone https://github.com/dicyy/docker-microservice
cd project
sudo docker compose up -d
```

## API
| Method | endpoint | Description |
|-----|-----|-----|
| GET | /users | get user |
| POST | /users | send user |
| EDIT | /users/index | edit user |
| DELETE | /users/index | delete user |
